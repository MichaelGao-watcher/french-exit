# File Cleaner - 文件清理工具

一个简单易用的桌面工具，帮你扫描电脑里的文件，按大小和类型筛选，然后选择性删除。

## 功能

- 选择任意目录进行扫描（默认用户目录）
- 按文件大小筛选（可设置最小 MB）
- 按文件扩展名筛选（如 `.mp4` `.zip`）
- 结果列表展示：文件名、大小、修改时间、完整路径
- 勾选要删除的文件，支持全选/取消全选
- 删除时**默认移到回收站**（安全可恢复）
- 扫描过程不卡界面（后台线程）

## 安装与运行

### 方式一：直接运行（推荐）

需要 Python 3.8 或更高版本。

```bash
# 1. 克隆或下载本项目
# 2. 进入目录
cd file-cleaner

# 3. （可选）安装 send2trash，删除时会移到回收站
pip install send2trash

# 4. 运行
python main.py
```

### 方式二：打包成 exe（无需安装 Python）

```bash
pip install pyinstaller
pyinstaller -F -w main.py
```

打包后 `dist/main.exe` 可以双击运行。

## 使用说明

1. 点击"浏览..."选择要扫描的目录
2. 设置最小文件大小（MB），留空或 0 表示不限制
3. （可选）输入文件扩展名过滤，如 `.mp4 .zip`，留空表示全部
4. 点击"开始扫描"
5. 在结果列表中点击第一列的复选框勾选要删除的文件
6. 点击"删除选中"，文件将被移到回收站

## 安全提示

- 扫描系统目录（如 `C:\Windows`）需要管理员权限，建议只扫描用户数据目录
- 删除前请仔细确认文件路径
- 建议先安装 `send2trash`，这样删除的文件会进入回收站，可随时恢复

## 技术栈

- Python 3.8+
- tkinter（标准库，无需安装）
- send2trash（可选，用于移到回收站）
