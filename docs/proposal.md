# French Exit — 需求提案文档

> 本文档记录 French Exit 工具的核心需求、用户画像、功能边界及待确认事项。
> 状态：✅ 阶段一已完成（需求已确认）

---

## 1. 产品一句话定义

French Exit 是一款面向**非技术背景职场人**的离职清理工具。

用户在离职前，只需要输入自己的**入职日期**，工具就会自动扫描电脑中从那天起产生的所有个人痕迹——包括聊天记录、软件登录信息、浏览器记录、系统日志等——然后让用户自行决定：**删除、保留，还是打包带走**。

| 维度 | 描述 |

| 技术水平 | 只会电脑基础操作（开软件、浏览网页、用 Office），**零编程背景** |
| 核心痛点 | 担心离职后个人聊天记录、私人文件、账号痕迹留在公司电脑上；手动清理太麻烦且容易漏 |
| 期望体验 | 简单、便捷、无感、体面——像平时用普通软件一样点点鼠标就能完成 |

#### 补充：GitHub CLI 与环境变量的处理（用户新增关注）

**能清除什么**：
- GitHub CLI 的本地配置文件（存储在 `C:\Users\你的用户名\AppData\Local\GitHub CLI\`）
- GitHub CLI 的登录凭证和缓存
- **用户级环境变量**中由 GitHub CLI 添加的 PATH 路径、GH_TOKEN、GITHUB_TOKEN 等

**风险与限制**：
- 如果是**系统级环境变量**（安装时选了"为所有用户安装"），普通权限下工具无法修改，需要用户手动处理
- 如果 GH_TOKEN 被**其他程序共用**（比如 Git、某个 IDE 插件），清除后可能导致其他工具失效
- PATH 环境变量如果删除了 GitHub CLI 路径，用户以后打开命令行就找不到 `gh` 命令了

**处理策略**：
- 工具会把 GitHub CLI 相关痕迹**列出来**，并明确标注风险："⚠️ 以下环境变量可能与其他工具共用，删除后 GitHub CLI 及相关工具将无法使用，确定清除吗？"
- 默认**不自动清除环境变量**，需要用户手动勾选确认
- 如果只清配置文件和登录状态（无风险），工具会标注为"安全清除"

## 5. 输入 / 输出清单

### 用户输入（用户能做什么）
1. 输入自己的**入职日期**（作为时间基准线）
2. **勾选/选择**要处理的痕迹类别（比如只扫聊天记录，不扫浏览器）
3. **随时暂停/继续**扫描任务
4. 对扫描结果中的每一条，决定是**删除、保留、还是打包**
5. 如果选择打包，指定**保存到哪里**（如 U 盘、移动硬盘、云盘文件夹）

### 系统输出（系统展示什么）
1. **扫描进度** — 告诉用户正在扫哪里，还剩多少，且**随时可暂停**
2. **结果清单** — 分类列出找到的所有痕迹，显示文件大小、位置、修改时间
3. **预览功能** — 让用户在删除前能点开看看内容是什么
4. **操作报告** — 处理完成后，生成一份清单，告诉用户删了什么、打包了什么
5. **完成庆祝页** — 一个精美的 HTML 页面，自动在浏览器打开

## 6. 待确认清单 ✅（请逐项确认后再继续）

### 6.1 关于"个人痕迹"的范围

- [x] **聊天记录**：
  - **网页版聊天工具**：技术上只能清除**浏览器本地痕迹**（Cookie、缓存、LocalStorage），这会让网页版退出登录，但**无法删除服务商服务器上的聊天记录**。工具会在扫描报告中明确标注这一点，并推荐用户"如需彻底清理网页版记录，请手动退出账号"。
  - **客户端聊天工具**：纳入扫描范围。
- [x] **浏览器书签**：用户自行判断。工具不直接处理书签，而是**弹出操作建议**："建议退出各浏览器账号同步，书签即可随账号移除"。
- [x] **个人文件（工作 vs 私人区分）**：用户承认区分困难。方案 —— **工具不按"工作/私人"区分，而是按类型列出**（文档、图片、视频、压缩包等），让用户自行勾选决定。
  - **例外：微信聊天记录** —— 微信聊天记录本身属于私人内容，**直接标记为建议处理（删除或打包），不需要逐条询问**。
  - **需要询问的范围**：仅限**桌面（Desktop）**和**下载文件夹（Downloads）**中的文件，按类型列出让用户自选。
- [x] **注册表和系统日志**：用户看不懂具体内容。方案 —— **工具自动按时间 + 名称 + 类型做推断**，只列出疑似包含个人信息的条目，并在每一项旁明确标注："⚠️ 此项由程序自动推断，请仔细确认后再操作"。

> 💡 **大白话解释**：网页版微信的聊天记录其实存在腾讯的服务器上，你电脑里只有浏览器的"记忆卡片"（Cookie）。我们能帮你把这些卡片烧了，让网页版失忆，但腾讯服务器上的记录我们碰不到。

### 6.2 关于"打包带走"的功能

- [x] 打包格式：统一压缩为 **`French-exit.zip`**，方便用户直接拷走。
- [x] 加密/无法打开的文件：由用户自行决定是否带走，但工具必须**明确弹窗提醒**："⚠️ 该文件已加密或无法直接打开，打包后可能无法查看，确定要带走吗？"

### 6.2a 关于"账号退出"（新增）

- [x] 工具**无法直接帮用户退出账号**（原因：网页端退出涉及浏览器操作和多端同步，客户端退出需要模拟点击且不稳定，容易出错）。
- [x] 替代方案：扫描完成后，工具会列出"你可能需要退出的账号清单"（如微信、企业微信、钉钉、QQ、浏览器账号、百度网盘、GitHub 等），每个账号旁提供**"点击跳转"**按钮，一键打开对应网站的"退出登录"或"账号安全"页面，用户手动操作即可。
- [x] 用户已接受此方案。

### 6.3 关于"删除"的安全程度

- [x] 默认采用**彻底删除**（安全擦除，不可恢复），但**必须在执行前弹窗征得用户同意**，明确告知："以下文件将被彻底删除，无法恢复，是否继续？"
- [x] 用户如果反悔，可以在最终确认清单中把某些项从"删除"改为"保留"或"打包"。

### 6.4 关于使用方式

> 💡 **AI 评估结论**：
> - **"纯 HTML 页面做交互，后台无感下载脚本"不可行**——浏览器安全沙箱禁止网页直接访问本地文件、注册表、系统日志等敏感区域。
> - **最便捷的形态**：**绿色免安装的单文件桌面程序**，双击即运行，不写入注册表，用完即走。
> - **技术路线建议**：UI 采用类网页的现代化技术（可实现苹果风格），底层打包为原生 .exe。推荐 **Tauri 框架**（体积小 ~10-20MB、性能高、绿色免安装、完全离线），或 **Electron**（体积大 ~100MB+，但生态成熟）。
> - **体验**：对用户来说就像一个精致的"原生 App"，外观像网页一样漂亮，但拥有完整的系统清理能力。

- [x] 形态：**绿色免安装**，双击运行，无需安装到系统。
- [x] 联网需求：**完全离线**，不需要网络即可运行所有功能。
- [x] 技术路线：**Tauri**（Rust + WebView2，体积小、启动快、性能高、完全离线）。用户已确认。

- [x] 资源占用策略：**默认自动限制**（CPU 不超过 30%，内存可控），界面不卡，不影响正常办公；同时提供**"解除限制"**选项，用户可手动开启全速扫描。

### 6.7 关于"完成庆祝页"（HTML）

- [x] 页面主文案：**"你已完成 French Exit，现在去享受生活吧"** — 用户已确认。
- [x] 小字标注内容：列出已完成的清理项目（如"已删除 1,247 条聊天记录，打包 3.2GB 个人文件至 D 盘"）—— 用户已确认此格式。
- [x] **自动弹出浏览器**：清理完成后，自动调用系统默认浏览器打开这个 HTML 页面 —— 用户已确认。

- [ ] HTML 页面保存位置：建议放在 **French-exit.zip 同目录**（即用户指定的打包输出目录），这样自毁时不会误删，用户也方便找到。如果用户没有打包，则默认放**桌面**。是否接受？

- [x] **零容忍误删**：所有删除操作（含彻底删除）都必须在执行前弹窗确认，且提供**最终确认清单**，用户逐项核对后才能执行。
- [x] **最后确认清单**：用户在点击"开始清理"前，必须看到一份完整的操作预览，包括：删什么、留什么、打包什么，支持最后一刻修改决定。

## 7. 术语表（大白话解释）

> 你在后续讨论中如果遇到不懂的词，可以在这里查。

| 术语 | 大白话解释 |
|------|-----------|
| **注册表** | Windows 系统的"记账本"，软件安装、用户配置都会记在里面。比如你装了微信，注册表里会留下你的微信安装路径、登录过的账号信息。 |
| **Cookie** | 浏览器的"记忆卡片"，网站靠它记住你是谁、是否登录过。比如你登录了淘宝，关掉浏览器再打开还是登录状态，就是 Cookie 的功劳。 |
| **日志（Log）** | 软件的"日记本"，记录了它什么时候启动过、出过什么错、用户做了什么操作。系统管理员可以通过日志了解你在电脑上做了什么。 |
| **缓存** | 软件的"临时仓库"，为了让你下次打开更快，把图片、网页内容等先存到本地。这些文件里可能包含你的私人信息。 |
| **彻底删除（安全擦除）** | 普通删除只是把文件标记为"可以覆盖"，实际数据还在硬盘上，用专业软件能恢复。彻底删除是把原数据反复覆盖，让任何人都恢复不了。 |
